# renraQR (VCard to QR Code)

連絡先を簡単に QR コードにできます。

できあがった QR コードをスマートフォンで読むと、簡単に連絡先に追加できます。やったぁ！

→ [QR コードを作ってみる](https://tools.sasakulab.com/renraqr/)
